Scan to open on a phone

Point a phone camera at the code to open the Observatory. On that phone, follow the install steps below to add it as an app.

✓ Installed. Look for the GBM Observatory icon on your home screen / app list.

  1. 1Open the link in Chrome (scan the QR code above, or tap the green button).
  2. 2Tap the ⋮ menu (top-right).
  3. 3Tap Install app (or Add to Home screen).
  4. 4The GBM Observatory icon appears in your app drawer — it opens full-screen like any app.
  1. 1Open the link in Safari (Apple requires Safari for installation).
  2. 2Tap the Share button (square with an upward arrow).
  3. 3Scroll down and tap Add to Home Screen.
  4. 4Tap Add — the icon appears on your home screen.
  1. 1Open the link in Chrome or Edge.
